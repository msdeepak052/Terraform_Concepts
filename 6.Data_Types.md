# üß© **Terraform Data Types**

Terraform supports **primitive**, **complex**, and **dynamic (any)** types.

---

## üü¢ **1. Primitive Types**

### **a. string**

Represents plain text.

```hcl
variable "instance_type" {
  type    = string
  default = "t2.micro"
}
```

**Usage:**

```hcl
resource "aws_instance" "example" {
  instance_type = var.instance_type
}
```

‚úÖ Example value: `"t2.micro"`
‚ùå Invalid value: `t2.micro` (without quotes)

---

### **b. number**

Represents numeric values (integer or float).

```hcl
variable "instance_count" {
  type    = number
  default = 3
}
```

‚úÖ Example value: `5` or `3.14`
‚ùå Invalid value: `"five"`

---

### **c. bool**

Represents true/false values.

```hcl
variable "enable_monitoring" {
  type    = bool
  default = true
}
```

‚úÖ Example values: `true`, `false`

---

## üü† **2. Complex (Collection) Types**

Collections hold **multiple values** of other types.

---

### **a. list(type)**

An **ordered collection** (like an array).

```hcl
variable "availability_zones" {
  type    = list(string)
  default = ["ap-south-1a", "ap-south-1b"]
}
```

**Access Example:**

```hcl
resource "aws_subnet" "example" {
  availability_zone = var.availability_zones[0]
}
```

‚úÖ Example value: `["us-east-1a", "us-east-1b"]`

---

### **b. set(type)**

An **unordered collection** of unique values.

```hcl
variable "allowed_ports" {
  type    = set(number)
  default = [22, 80, 443]
}
```

**Difference from list:**

* No duplicates
* Order not guaranteed

---

### **c. map(type)**

A **key-value** collection.

```hcl
variable "region_amis" {
  type = map(string)
  default = {
    ap-south-1 = "ami-12345"
    us-east-1  = "ami-67890"
  }
}
```

**Access Example:**

```hcl
ami = var.region_amis["ap-south-1"]
```

‚úÖ Think: like a Python dictionary or JSON object.

---

### **d. object({ attributes... })**

A **collection of named attributes** ‚Äî like a structured record.

```hcl
variable "instance_config" {
  type = object({
    instance_type = string
    volume_size   = number
    monitoring    = bool
  })
  default = {
    instance_type = "t2.micro"
    volume_size   = 20
    monitoring    = true
  }
}
```

**Usage:**

```hcl
instance_type = var.instance_config.instance_type
```

‚úÖ Think: like a JSON object with fixed schema.

---

### **e. tuple([types...])**

An **ordered collection** of values with possibly **different types**.

```hcl
variable "example_tuple" {
  type    = tuple([string, number, bool])
  default = ["t2.micro", 2, true]
}
```

‚úÖ Think: like a fixed-length array with mixed data types.

---

## üîµ **3. Dynamic Type ‚Äî `any`**

If you don‚Äôt want to restrict the type (Terraform will auto-detect):

```hcl
variable "input_value" {
  type = any
}
```

You can assign **string**, **list**, **map**, or anything ‚Äî Terraform will infer it at runtime.

---

# üìö **Summary Table**

| Type                | Description                 | Example Value                   |
| ------------------- | --------------------------- | ------------------------------- |
| `string`            | Single line of text         | `"t2.micro"`                    |
| `number`            | Numeric value               | `3`                             |
| `bool`              | Boolean value               | `true`                          |
| `list(string)`      | Ordered list                | `["a", "b"]`                    |
| `set(number)`       | Unordered unique collection | `[22, 80]`                      |
| `map(string)`       | Key-value pairs             | `{ env = "dev" }`               |
| `object({})`        | Structured record           | `{ name = "deepak", age = 30 }` |
| `tuple([types...])` | Fixed ordered values        | `["deepak", 30, true]`          |
| `any`               | Any value                   | `"anything"`                    |

---

# üß† **Bonus ‚Äî Type Constraints**

You can nest types too!

Example:

```hcl
variable "servers" {
  type = list(object({
    name = string
    size = string
  }))
  default = [
    { name = "web", size = "t2.micro" },
    { name = "db",  size = "t2.medium" }
  ]
}
```

# üß© **Example Full Usage**

**variables.tf**

```hcl
variable "instance_type" {
  type    = string
  default = "t2.micro"
}

variable "instance_tags" {
  type = map(string)
  default = {
    Name        = "web-server"
    Environment = "dev"
  }
}
```

**main.tf**

```hcl
resource "aws_instance" "example" {
  ami           = "ami-0c55b159cbfafe1f0"
  instance_type = var.instance_type
  tags          = var.instance_tags
}
```
